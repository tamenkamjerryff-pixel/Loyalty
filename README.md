# Loyalty

A simple repository for the **Loyalty** project.

## Overview

This repository hosts the source code and documentation for the Loyalty project.
It is currently in its initial setup phase.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/tamenkamjerryff-pixel/Loyalty.git
cd Loyalty
```

## Project Setup (for reference)

These are the commands that were used to bootstrap this repository:

```bash
echo "# Loyalty" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/tamenkamjerryff-pixel/Loyalty.git
git push -u origin main
```
